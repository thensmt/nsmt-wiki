# Design Tokens — Cross-System Comparison

Two design systems serve different NSMT products. This page compares their tokens side-by-side.

## Token Comparison

| Token | NSMT Fastbreak (Broadcast) | Kinetic Championship (Ticketing) |
|---|---|---|
| Primary Color | `#0E80FC` (NSMT Blue) | `#552583` (Deep Purple) |
| Background | Black | `#131313` (Near Black) |
| Accent | White | `#ecbf7b` (Gold) |
| Headline Font | Space Grotesk | Lexend |
| Body Font | Inter | Manrope |
| Label Font | — | Space Grotesk |
| Icon System | Material Symbols | — |
| Border Radius | Varies | 0px everywhere |
| Shadows | Varies | None |
| CSS Framework | Tailwind CDN | Tailwind |

## Shared DNA

Both systems share common traits despite serving different products:

- **Space Grotesk** appears in both — as the headline font in Fastbreak, and as the label font in Kinetic Championship
- **Tailwind** is the CSS framework for both
- **Dark-themed** — both default to dark backgrounds (pure black vs. near-black)
