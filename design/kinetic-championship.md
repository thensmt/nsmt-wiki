# Kinetic Championship Design System

**Used in:** The Fastbreak App (`thensmt/little-rock`)
**Origin:** "The Kinetic Championship" concept via Google Stitch export
**Stitch export:** `~/Downloads/Stitch/stitch_my_tickets_the_arena_concept/`
**Aesthetic:** Dark arena

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Deep Purple | `#552583` / `#ddb8ff` | Primary (dark/light variants) |
| Black | `#131313` | Background (near-black) |
| Silver | `#c6c6c6` | Secondary text, dividers |
| Gold | `#ecbf7b` | Accent, highlights |

## Typography

| Role | Font | Style |
|------|------|-------|
| Headlines | Lexend | Bold + Italic |
| Body text | Manrope | Regular |
| Labels | Space Grotesk | Regular |

## Design Rules

- **Border radius:** 0px everywhere — sharp corners only
- **Shadows:** None — depth communicated through tonal layering
- **Depth model:** Tonal depth layering using the color palette (lighter surfaces = closer)
- **CSS Framework:** Tailwind
