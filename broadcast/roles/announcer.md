# Announcer

## Play-by-Play — Nels

**Required for minimum crew:** Yes
**Equipment:** [Audio-Technica broadcast headset](../equipment/audio.md)
**Can double as:** Nothing — voice is always live

### Responsibilities
- Game narration — calls the action as it happens
- Sets the pace and energy of the broadcast
- Introduces segments, reads sponsor mentions
- Throws to color commentator and sideline reporter

---

## Color Commentator — Dwight

**Required for minimum crew:** No (ideal, not required)
**Equipment:** [Audio-Technica broadcast headset](../equipment/audio.md)
**Can double as:** Stat tracker in a pinch (not recommended)

### Responsibilities
- Analysis and context between plays
- Storytelling — player backgrounds, matchup breakdowns
- Fills dead-ball time with insight
- Engages in back-and-forth with play-by-play

---

## Sideline Reporter

**Required for minimum crew:** No (optional)
**Equipment:** [Sennheiser guest headset](../equipment/audio.md)

### Responsibilities
- Post-game player/coach interviews
- Halftime segments
- Injury updates, sideline observations

---

## Minimum Crew Note

The broadcast can operate with a single announcer (Nels on play-by-play). Adding Dwight as color commentator significantly improves the product. Sideline reporter is a nice-to-have for post-game content.

## Setup Checklist
1. Plug headsets into [Rodecaster Pro I](../equipment/audio.md) channels
2. Set input levels for each mic (speak at normal broadcast volume)
3. Confirm monitoring — announcers should hear program audio in their headsets
4. Do a mic check with [TD](tech-director.md) — verify audio is reaching OBS

## During Broadcast
- Keep consistent distance from mic
- Watch for [TD](tech-director.md) cues (talkback system TBD — see [Audio](../equipment/audio.md))
- During commercial breaks, stay ready for quick return

## Common Issues
- **Audio clipping:** Reduce Rodecaster input gain, check OBS audio filters (Compressor, Noise Gate, Limiter — in that order)
- **Can't hear program audio:** Check Rodecaster headphone routing
- **No talkback from TD:** This system is TBD — see [Audio](../equipment/audio.md)
