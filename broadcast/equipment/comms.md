# Comms — Hollyland Solidcom SE

**Model:** Hollyland Solidcom SE
**Qty:** 5-user wireless intercom system
**Used by:** [Tech Director](../roles/tech-director.md), [Camera Operators](../roles/camera-operator.md), production crew

## Specs
- 5 single-ear headsets (1 hub + 4 remote)
- 2.4 GHz wireless
- 1,100 ft range
- Full duplex communication
- ENC (Environmental Noise Cancellation)
- PTT (Push-to-Talk) mute button

## Configuration
- Used for production crew to camera operator communication
- TD calls camera switches and cues via intercom
- Camera operators confirm shots and report issues

## Known Issues

**Batteries died during Hoopfest (2026-03-22).** This was the most significant equipment failure during the first NSMT broadcast. Spare batteries or a charging rotation must be planned for every future event.

See [Power](power.md) for the spare battery action item and [Hoopfest event report](../events/hoopfest-2026-03-22.md) for full details.

### Troubleshooting
- **Dropouts:** Check battery level first, then check range (1,100 ft max), then check 2.4 GHz interference from other devices/Wi-Fi
- See [Troubleshooting runbook](../runbooks/troubleshooting.md) for more
