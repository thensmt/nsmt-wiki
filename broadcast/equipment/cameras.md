# Cameras

## Top Camera — Sony FX3

**Model:** Sony FX3 (ILME-FX3)
**Qty:** 2 (owned)
**Used by:** [Top Camera Operator](../roles/camera-operator.md)

### Specs
- Full-frame mirrorless cinema camera
- Mounted on SmallRig AD-100 FreeBlazer tripod (78" max height, 22 lb capacity)
- Elevated position for hero/wide shot

### Lens Options

| Lens | Price | Notes |
|---|---|---|
| Sony 24-70mm f/2.8 GM II | ~$2,300 | Standard zoom, sharp across range |
| Sony 28-70mm f/2 GM | ~$2,900 | One stop faster — better for dim gyms |

The f/2 lens is recommended if budget allows, since many school gyms have poor lighting.

### Power
- NP-FZ100 battery (native)
- V-mount to NP-FZ100 adapter available for extended runtime
- Spare NP-FZ100 batteries recommended
- See [Power](power.md) for V-mount standardization

## Baseline Cameras — Decision Pending

Three options under evaluation for the baseline (courtside) camera positions.

### Comparison Table

| | **Option A: Sony PXW-Z200** | **Option B: Sony HXR-NX800** | **Option C: All FX3** |
|---|---|---|---|
| **Price** | ~$4,000 each | ~$3,300 each | Already owned (need lenses) |
| **Zoom** | 20x optical | 20x optical | Limited by lens choice |
| **Outputs** | SDI + HDMI | HDMI only | HDMI |
| **Autofocus** | AI AF | AI AF | AI AF |
| **ND Filters** | Built-in | Built-in | None (external needed) |
| **Form Factor** | Camcorder | Camcorder | Mirrorless (shoulder rig) |
| **Recommendation** | **Recommended** | Budget option | Limits zoom range |

**Option A (PXW-Z200) is recommended** — the SDI output provides redundancy, built-in ND filters simplify outdoor shoots, and the 20x zoom is essential for baseline coverage.

### Baseline Camera Rig
- SmallRig Classic 4480 shoulder rigs x2 for baseline cameras
- Rolling chairs for camera operators (mobility along baseline)
- See [Camera Operator](../roles/camera-operator.md) for operating details

### Power (Camcorder Route)
- V-mount to BP-U30 adapter plates needed (KOCACKOO or Alvins brands)
- Dummy battery box is 38mm tall — **verify clearance fits in battery compartment** before purchasing
- See [Power](power.md) for full battery inventory

## Known Issues
- Zoom range is the main limitation if going all-FX3 (Option C) — no lens matches 20x optical zoom of camcorders
- BP-U30 adapter plate fitment is unverified — needs physical test
