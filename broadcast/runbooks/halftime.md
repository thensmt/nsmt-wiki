# Halftime Runbook

---

## Transition to Halftime

1. Switch to Pre-Game/Halftime scene (**F2**)
2. Mute broadcast audio if no halftime commentary is planned
3. [Announcers](../roles/announcer.md) can recap first half or go off-mic

---

## Equipment Checks

Use halftime to check equipment before it becomes a problem in the second half:

- [ ] Check battery levels on all [V-mount batteries](../equipment/power.md) — swap any below 30%
- [ ] Check [Hollyland Solidcom SE](../equipment/comms.md) headset battery levels — **this failed during Hoopfest** (see [event report](../events/hoopfest-2026-03-22.md))
- [ ] Check Teradek TX/RX signal stability
- [ ] Check OBS Stats dock — any dropped frames or encoding lag trends?
- [ ] Verify recording is still active (Ctrl+Shift+R indicator)

---

## Stat Tracker Coordination

- [ ] Confirm second half rosters/lineups with [Stat Tracker](../roles/stat-tracker.md)
- [ ] Verify halftime score is correct in scorebug
- [ ] Reset any per-half stats if applicable
- [ ] Check that overlay connection is still live

---

## Resume Broadcast

1. TD announces "back in 60 seconds" on [intercom](../equipment/comms.md)
2. Camera ops return to positions
3. [Announcers](../roles/announcer.md) ready
4. TD counts down: "Back in 10... 5, 4, 3, 2, 1"
5. Switch back to NSMT Game scene (**F1**)
6. Broadcast resumes
