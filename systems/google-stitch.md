# Google Stitch

**Service:** Google Stitch -- UI design tool
**Used by:** NSMT design workflow

## What It Does

Google Stitch is used to create UI designs, which export as HTML, CSS, and Markdown files. David designs in Stitch, then brings exports into Claude Code for implementation.

## Export Location

All Stitch exports land in `~/Downloads/Stitch/`.

## Known Exports

| Export | Used For |
|--------|----------|
| `stitch_scorebug_front_end_overlay.zip` | Scorebug overlay design for [Fastbreak V2](../projects/fastbreak-v2.md) |
| `stitch_my_tickets_the_arena_concept/` | "Kinetic Championship" ticketing app design for [The Fastbreak App](../projects/fastbreak-app.md) |

## Workflow

1. David designs in Google Stitch
2. Exports land in `~/Downloads/Stitch/`
3. Exports are brought into Claude Code for implementation
4. When David mentions "Stitch designs" or asks to use Google Stitch, check `~/Downloads/Stitch/` for new exports
