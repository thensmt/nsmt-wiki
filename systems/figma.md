# Figma

**Service:** Figma -- collaborative design tool
**Connected since:** 2026-03-26
**Used by:** NSMT frontend design

## What It Does

Figma is available for frontend design work across NSMT projects. Connected programmatically via MCP servers for design-to-code and code-to-design workflows.

## MCP Integrations

- **TalkToFigma MCP server** -- read/write Figma files programmatically (create frames, text, shapes, export images, manage components)
- **Pencil MCP server** -- design .pen files with structured operations

## Use Cases

- Frontend design for broadcast system (scorebug, producer, iPad, overlay)
- Component library and design system creation
- Prototyping new layouts before code implementation
- Design-to-code: implement designs from Figma into production code
- Code-to-design: generate Figma representations from existing code
